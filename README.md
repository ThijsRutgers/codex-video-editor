# 🎬 Video Editor Studio

Autonomous AI video editor. Drop in a video, type one prompt, get an edited video back.

## Quick Start

```bash
# 1. Setup (once)
chmod +x setup.sh && ./setup.sh

# 2. Add your video
cp ~/Downloads/my-video.mp4 public/video.mp4

# 3. Start Remotion Studio (terminal 1)
npm run dev

# 4. Start Codex (terminal 2)
codex

# 5. Paste your prompt (see examples below)
```

## Example Prompts

### Basic edit
```
Edit public/video.mp4.
Speakers: Jan de Vries (CEO), Lisa Bakker (CTO).
Add: opening title "Innovation Summit 2026", name cards, captions, outro with logo.
Style: professional, dark tones.
Follow the full workflow from CODEX.md. Work autonomously.
```

### Minimal (let AI decide)
```
Edit public/video.mp4. Add professional overlays: title, name cards for
all speakers (detect from transcript), subtitles, and an outro.
Follow CODEX.md. Work autonomously. Report when done.
```

### Specific timing
```
Edit public/video.mp4.
- 0:00-0:04: Title "Product Launch 2026"
- 0:05: Name card "Sarah Chen, VP Product"
- 0:45: Name card "Mike Ross, Engineering Lead"
- Throughout: TikTok-style captions
- Last 5 seconds: Logo (public/assets/logo.png) + "acme.com"
Follow CODEX.md.
```

## Customization

### Change the visual style
Edit the **Style Guide** section in `CODEX.md`. Change colors, fonts,
animation settings, positioning — Codex reads this on every prompt.

### Add components
Create new `.tsx` files in `src/components/` and register them in
the components table in `CODEX.md`.

### Run fully autonomous (Ralph loop)
```bash
# Install Ralph (once)
git clone https://github.com/d0ublecl1ck/ralph-for-codex.git /tmp/ralph
cp /tmp/ralph/ralph.sh scripts/
chmod +x scripts/ralph.sh

# Edit prd.json with your specific video details, then:
./scripts/ralph.sh 15
```
